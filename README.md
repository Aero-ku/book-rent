## 体验步骤
### 1、安装 wepy脚手架
```
npm install wepy-cli -g
```
### 2、下载源代码
```
git clone https://github.com/Aero-ku/book-rent.git
```

### 3、 安装开发依赖
npm install

### 4、运行
```
npm run dev
```
### 5、导入至微信开发者工具
编译完成后会生成dist目录，开发者工具本地开发目录选择打包编译后生成的dist目录即可浏览该项目

### 6、注意事项
微信开发者工具点击详情 - > ES6 转ES5项 关闭 - > 不校验合法域名项  打开
